## 🛠️ Keunggulan Tech Stack

Proyek ini dibangun menggunakan kombinasi teknologi terbaik untuk memastikan kecepatan, skalabilitas, dan estetika:

-   **[React 18](https://reactjs.org/)**: Memanfaatkan fitur *concurrent rendering* terbaru untuk UI yang sangat responsif.
-   **[Vite](https://vitejs.dev/)**: Workflow pengembangan kilat dan bundel produksi yang sangat optimal.
-   **[Framer Motion](https://www.framer.com/motion/)**: Standar industri untuk animasi web yang terasa hidup dan nyata.
-   **[Supabase](https://supabase.com/)**: Backend berbasis PostgreSQL yang tangguh, siap untuk manajemen data dinamis dan keamanan tingkat enterprise.
-   **[PM2](https://pm2.io/)**: Manajemen proses tingkat lanjut untuk memastikan layanan tetap online 24/7 dengan *hot reload* tanpa downtime.
-   **[Lucide React](https://lucide.dev/)**: Set ikon modern yang ringan dan konsisten.
-   **Vanilla CSS**: Sistem desain kustom menggunakan *CSS Variables* untuk fleksibilitas maksimal tanpa beban library tambahan.

---

## 🚀 Mengapa Menggunakan Stack Ini?

1.  **Performa Maksimal**: *Overhead* minimal memastikan waktu muat yang instan.
2.  **Skalabilitas**: Arsitektur komponen yang modular memudahkan pengembangan fitur di masa depan.
3.  **Presisi Estetika**: Kontrol penuh atas sistem desain memungkinkan terciptanya antarmuka yang unik dan tidak generik.
4.  **Reliabilitas**: Eksekusi proses yang dikelola PM2 menjamin platform tetap stabil bahkan setelah restart server.

## 📦 Memulai (Quick Start)

### Prasyarat
-   Node.js (v16 atau lebih tinggi)
-   npm atau yarn

### Instalasi

1.  Clone repositori:
    ```bash
    git clone https://github.com/syarfandi/syarfandi.github.io.git
    ```

2.  Instal dependensi:
    ```bash
    npm install
    ```

3.  Jalankan dalam mode pengembangan:
    ```bash
    npm run dev
    ```

4.  Build untuk produksi:
    ```bash
    npm run build
    ```
